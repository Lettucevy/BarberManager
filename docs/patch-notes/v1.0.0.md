# Patch notes - v1.0.0 (planejada)

**Data de lancamento prevista:** 2a quinzena de julho de 2026
**Tag Git:** `v1.0.0`
**Status:** planejamento

## Resumo

Primeira versao "estavel" do produto, com integracoes externas e
suporte a pagamento. Indicada para uso em producao por uma unica
barbearia piloto.

## Mudancas previstas

### Pagamento
- Integracao com gateway (Mercado Pago ou PagSeguro) no checkout.
- Sinal de 20% para confirmar agendamento.
- Comprovante enviado por email.

### Notificacoes
- Confirmacao do agendamento por WhatsApp via API oficial ou
  provedor intermediario.
- Lembrete 24h antes do horario.
- Mensagem automatica quando o cliente cancela.

### Aplicativo
- Manifesto PWA completo, instalavel no celular.
- Tela de splash com a marca da barbearia.
- Modo offline para consulta de horarios disponiveis.

### Operacional
- Migracao opcional para PostgreSQL (docker-compose).
- Backup automatico diario do SQLite.
- Logs estruturados com `pino`.

### Painel
- Edicao de agendamento (reagendamento com 1 clique).
- Relatorio semanal em PDF para o dono.
- Controle de comissoes dos profissionais.

## Risco
- A escolha do gateway de pagamento precisa de CNPJ e conta
 商务. Caso nao esteja disponivel em julho, pagamento fica para
  v1.1.0.

## Proxima versao

  fidelidade (pontos a cada corte), e analise de mix de servicos
  por periodo.
